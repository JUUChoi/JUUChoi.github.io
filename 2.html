[R] Errors in Extrafont Package

Posted by JWChoi on October 22, 2021

When using an extrapont package in Windows, an error
"font family not found in Windows font database"
may occur. I tried font_import() because I thought there was a problem with the front import, but the message No FontName. Skipping popped up.

Through googling, I found that this problem is traced to the Rttf2pt1 package and downgrading that to v1.3.8 can be the solution. But this method didn't work very well for me either.

The most effective way so far was to add a line of code like
windowsFonts("KoPubWorldDotum Bold" = windowsFont("KoPubWorldDotum Bold")).
If you know a better way, please share it with me.
